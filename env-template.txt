# =====================================================
# SETUP INSTRUCTIONS
# =====================================================

# 1. IGDB SETUP:
#    - Go to https://api.igdb.com/
#    - Sign up for a free account
#    - Create a new application
#    - Copy Client ID and Client Secret
#    - Note: IGDB is free for commercial use

# 2. GOOGLE SEARCH SETUP:
#    - Go to https://programmablesearchengine.google.com/
#    - Create a new search engine
#    - Add the gaming wiki domains from our curated list
#    - Get the Search Engine ID
#    - Enable the Custom Search API in Google Cloud Console
#    - Get the API Key

# 3. YOUTUBE SETUP (Phase 3):
#    - Go to https://console.cloud.google.com/
#    - Enable YouTube Data API v3
#    - Create credentials (API Key)

# 4. REDDIT SETUP (Phase 3):
#    - Go to https://www.reddit.com/prefs/apps
#    - Create a new application
#    - Choose "script" as the application type
#    - Copy Client ID and Client Secret

# TROUBLESHOOTING
# =====================================================

# Common Issues:
# - "IGDB credentials not configured" → Check IGDB_CLIENT_ID and IGDB_CLIENT_SECRET
# - "Google Search credentials not configured" → Check GOOGLE_SEARCH_API_KEY and GOOGLE_SEARCH_ENGINE_ID
# - "Rate limit reached" → Wait a moment, the app respects API limits
# - "Authentication failed" → Verify your API keys are correct
