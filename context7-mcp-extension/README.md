# Context7 MCP Extension

This extension integrates with the Context7 MCP server to provide real-time documentation access within your development environment.

## Features

- Real-time access to documentation from the Context7 MCP server.
- Command palette integration for fetching and resolving library IDs.
- Contextual documentation display based on the current code context.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/context7-mcp-extension.git
   ```
2. Navigate to the project directory:
   ```
   cd context7-mcp-extension
   ```
3. Install the dependencies:
   ```
   npm install
   ```
4. Open the project in your code editor.

## Usage

- Open the command palette (Ctrl+Shift+P or Cmd+Shift+P).
- Type `MCP: Fetch Documentation` to retrieve documentation for the current library.
- Use `MCP: Resolve Library ID` to resolve the library ID based on the current context.
