Otagon: AI-Powered Game Assistant & Screenshot Platform
Otagon is a next-generation AI companion for gamers, designed to enhance your gaming experience with real-time insights, hands-free voice responses, and seamless screenshot management. Built with modern web technologies and powered by advanced AI models, Otagon helps you stay focused, improve your gameplay, and easily share your best moments.

üõ†Ô∏è How Was Otagon Built?
1. Frontend: React, TypeScript, Vite, Tailwind CSS for a fast, responsive UI.
2. Backend: Supabase for authentication, real-time database, and secure storage.
3. AI Integration: Google Gemini API for advanced language and vision capabilities.
4. WebSocket Relay: Real-time communication between PC and mobile for instant screenshot transfer and AI chat.

6. Performance: CDN-backed storage, optimized image handling, and minimal database footprint.
